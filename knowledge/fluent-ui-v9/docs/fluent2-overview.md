# Fluent 2 Design System — Home

> Fluent 2 is Microsoft's design system for digital products and experiences. It is reflected in working code, UI kits, and guidelines that work seamlessly across apps and devices.

## Pillars

- **Design** — Figma UI kits with design assets that map directly to the code libraries. → [`/get-started/design`](https://fluent2.microsoft.design/get-started/design)
- **Develop** — Installation guides + tooling tips for each platform library. → [`/get-started/develop`](https://fluent2.microsoft.design/get-started/develop) (snapshotted as [`fluent2-develop.md`](./fluent2-develop.md))
